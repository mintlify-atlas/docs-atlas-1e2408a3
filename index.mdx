---
title: "Welcome to Sinatra"
description: "A minimal Ruby DSL for quickly building web applications and APIs with routing, templates, middleware, and extensions."
---

Sinatra is a lightweight Ruby framework that lets you define routes, render templates, and handle HTTP requests with minimal boilerplate. Whether you're building a microservice, a REST API, or a full web app, Sinatra gets you running in minutes.

```ruby
require 'sinatra'

get '/' do
  'Hello world!'
end
```
